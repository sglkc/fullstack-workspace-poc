# TypeScript Workspace POC

## Use pnpm!

## Front-end

Vite, Preact, UnoCSS, React Router, Axios

## Back-end

Fastify, SQLite 3, Kysely, Nodemon

## How to run

1. Copy `.env.example` to `.env`
2. Set environment variables
3. Dev server
   ```sh
   pnpm dev
   ```
4. Build
   ```sh
   pnpm build
   ```
5. Production
   ```sh
   pnpm start
   ```
